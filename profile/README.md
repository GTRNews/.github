# GTRNews


GTRNews is a modern, scalable news platform built with a strong focus on performance, reliability, and automation. This organization contains all core services, infrastructure, and tooling required to develop and operate the platform.

---

## What is GTRNews?

GTRNews is designed as a real-time news delivery system with a modular architecture and production-grade DevOps practices. The goal is to provide a highly reliable and observable system from the early stages of development.

---

## Tech Stack

* **Frontend**: Next.js (TypeScript)
* **Backend**: Python (FastAPI)
* **Database**: MongoDB (Azure Cosmos DB)
* **Cloud**: Microsoft Azure (Web Apps, Container Apps)
* **CI/CD**: GitHub Actions
* **Monitoring**: Sentry + logging + metrics

---

## Architecture

* Service-oriented, modular design
* Clear separation between frontend, backend, and infrastructure
* Built for scalability and maintainability
